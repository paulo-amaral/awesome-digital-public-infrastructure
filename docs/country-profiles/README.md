# Country DPI Profiles

Detailed profiles of national Digital Public Infrastructure implementations. Each profile documents the DPI stack, open-source tools in use, key milestones, and lessons learned.

## Available Profiles

| Country | DPI Highlights | Status |
|---------|----------------|--------|
| [India](india.md) | Aadhaar, UPI, ONDC, DigiLocker, Ayushman Bharat | Complete |
| [Estonia](estonia.md) | X-Road, e-Residency, KSI Blockchain, i-Voting | Complete |
| [Brazil](brazil.md) | PIX, GOV.BR, Open Finance, e-Title | Complete |
| [Timor-Leste](timor-leste.md) | CPMIS, e-SISCA, RDTL digital services | Complete |
